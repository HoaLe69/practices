# AWS VPC Basic

> Provision a minimal AWS VPC with public and private subnets using Terraform.

## Goal

Learn core Terraform workflow (`init` / `plan` / `apply` / `destroy`) and how a VPC,
subnets, route tables, and an internet gateway fit together.

## Prerequisites

- Terraform 1.6+
- AWS credentials configured (`aws configure`)

## How to run

```bash
terraform init
terraform plan
terraform apply
# ... explore ...
terraform destroy
```
